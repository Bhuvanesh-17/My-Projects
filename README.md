Blink It Sales Analysis Report
--------------------------------
Description:
- Designed an interactive Power BI dashboard to analyze Blinkit sales performance across outlet location, outlet size, item type, and outlet type.
- Tracked key KPIs such as Total Sales, Average Sales, Number of Items, and Average Rating.
- Analyzed sales trends by outlet establishment year to understand business growth over time.
- Performed category-wise analysis based on item type and fat content to identify customer purchase patterns.
- Compared Tier-wise outlet performance to evaluate regional demand and sales contribution.
---------------------------------
Technologies used in this Project: 
- Microsoft Power BI
- Microsoft Excel
